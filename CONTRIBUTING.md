# Contributing to the Identity Graph

Thank you for contributing to the modular identity graph for Lee Richard Grimwade / SmiLee.  
This repository maintains a structured, machine‑readable knowledge graph using JSON‑LD and HTML wrappers.

## 📁 Folder Structure
- All JSON‑LD entities live in `/jsonld/`
- Each `.jsonld` file has a matching `.html` wrapper (auto‑generated)
- `index.html` contains the primary identity
- `sitemap.xml` is auto‑generated
